# 🎮 坦克大战 (Tank Battle)

经典 FC 红白机风格坦克大战，纯 HTML5 Canvas 实现，无需任何依赖，打开即玩！

## 🕹️ 操作说明

| 按键 | 功能 |
|------|------|
| ⬆⬇⬅➡ / WASD | 移动坦克 |
| 空格键 | 射击 |
| P | 暂停 / 继续 |
| R | 重新开始 |

## ✨ 游戏特色

- 🎯 **经典玩法** - 保卫基地，消灭所有敌方坦克
- 🧱 **可破坏地形** - 砖墙可被子弹摧毁，钢板不可破坏
- 🗺️ **多关卡** - 内置 3 个不同布局的关卡
- 🎁 **道具系统** - 击杀敌人随机掉落道具（加速、护盾、全屏炸弹、冻结）
- 💥 **粒子特效** - 爆炸粒子动画
- 🔊 **音效** - Web Audio API 生成的复古音效
- 📱 **响应式** - 适配不同屏幕尺寸

## 🚀 快速开始

1. 克隆仓库：
```bash
git clone https://github.com/YOUR_USERNAME/tank-battle.git
```

2. 直接用浏览器打开 `index.html` 即可开始游戏！

或者直接访问 GitHub Pages 在线游玩。

## 🛠️ 技术栈

- HTML5 Canvas
- 原生 JavaScript (ES6+)
- Web Audio API
